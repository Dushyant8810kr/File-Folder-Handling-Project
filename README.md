# 📁 Smart File Manager

A simple File Manager Web App built using Streamlit.

## 🚀 Live Demo

👉 https://file-handling-project-5pmgrrhupyrybyzchkjbmj.streamlit.app/

## ✨ Features

* Create & delete folders
* Create, read, update files
* View all files in directory
* Simple and clean UI

## 🛠 Tech Stack

* Python
* Streamlit

## 📦 Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
